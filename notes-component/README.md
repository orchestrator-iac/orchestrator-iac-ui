# Notes Component

This project is a simple notes application built with React and TypeScript. It allows users to create, view, and manage notes. The application consists of several components that work together to provide a seamless user experience.

## Project Structure

- `src/components/NotesList.tsx`: A component that renders a list of notes.
- `src/components/NoteCard.tsx`: A component that displays the details of a single note.
- `src/components/AddNoteForm.tsx`: A component that provides a form for adding new notes.
- `src/types/Note.ts`: TypeScript interface defining the structure of a note object.
- `src/styles/notes.styles.ts`: Styles for the notes components.
- `src/utils/noteHelpers.ts`: Utility functions for handling notes.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd notes-component
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

## Usage

- Use the **Add Note** form to create new notes.
- View your notes in the list below the form.
- Each note displays its title and content.
